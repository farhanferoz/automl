Training Objective
Total Loss (trained end-to-end)
L = Lregression + Lclassification
Regression Loss — Gaussian Likelihood
Assume p(y | x) = N(y ; ŷ, σ²)
Maximise log-likelihood ⟹ minimise negative log-likelihood:
Lreg = − log p(y | x) = ½ [ log(2π) + log σ² + (y − ŷ)² / σ² ]
Trains the model to produce accurate predictions AND calibrated uncertainty
Classification Loss — Cross-entropy on discretised targets
Lcls = − ∑j=1k  𝟙[y ∈ binj] · log pj(x)
Ensures the classifier assigns y to the correct bin (quantile-based boundaries)
Lcls ensures good class assignments   |   Lreg ensures accurate predictions & calibrated uncertainty
